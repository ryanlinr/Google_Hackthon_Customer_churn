# Team Reports

This directory contains all team reports related to the Google Hackathon Customer Churn project.

## Structure

Upload your PDF reports here. You can organize them by:
- Team member name
- Report type (e.g., analysis, findings, recommendations)
- Date or milestone

## Naming Convention

Please use descriptive names for your reports, for example:
- `customer_churn_analysis_report_YYYY-MM-DD.pdf`
- `data_insights_report_[team_member].pdf`
- `final_recommendations_report.pdf`

## Guidelines

- All reports should be in PDF format
- Include the date in the filename when applicable
- Use clear, descriptive names
- Add a brief description below when uploading a new report
